Creazione di regioni

3 di 10

Per creare delle regioni:

  1. Nella pila strumenti selezionare lo strumento a cui si desidera applicare la regione o aggiungerne uno nuovo.
  2. Fare clic sul pulsante Regioni nella barra degli strumenti. Il pulsante del tipo di regione sarà visualizzato a destra della barra degli strumenti.
  3. Nella barra degli strumenti fare clic sul pulsante della regione che si desidera creare: poligonale, curvilinea o Bézier.
  4. Muovere il mouse sulla foto e fare clic nel punto in cui si desidera creare un punto di controllo. Ripetere questo passaggio per ogni punto di controllo che si desidera creare.
  5. Fare doppio clic per creare l'ultimo punto di controllo.

Nota: se inavvertitamente si inizia a creare una regione, premere il tasto ESC (escape) per annullare l'azione.
