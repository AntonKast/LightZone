Sfocatura gaussiana

Lo strumento Sfocatura gaussiana consente di sfocare alcune o tutte le parti di una foto in base al raggio specificato. Questo strumento può essere usato per creare un effetto sfocato o per correggere imperfezioni o sgranature.

Per sfocare una foto:

  1. Fare clic sul pulsante dello strumento di sfocatura nella barra degli strumenti. Sarà visualizzato un nuovo strumento di sfocatura in cima alla pila strumenti.
  2. Usare il dispositivo di scorrimento Raggio per regolare l'effetto di sfocatura:
    • Il dispositivo di scorrimento Raggio consente di controllare l'ampiezza dell'effetto sfocato intorno a ogni pixel.
  3. In alternativa:
    • Creare una maschera usando le regioni.
    • Selezionare una modalità fusione diversa.
    • Regolare il dispositivo di scorrimento Opacità.
    • Effettuare una selezione di luminosità/colore.
