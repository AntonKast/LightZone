Styles : Création

3 de 6

Créer un style enregistre une chaîne de développement avec un nom pour une application future à d'autres photos. De plus, vous pouvez remplacer la Courbe de Tonalité par défaut d'un appareil avec un style que vous créez.

Pour créer un style :

  1. Créez une chaîne de développement à votre goût.
  2. Choisissez Fichier > Enregistrer comme un Style...
  3. Entrez un nom pour le style.
  4. Entrez ou choisissez un nom pour le dossier du style. (Les Styles sont groupés dans des dossiers de votre choix.)
  5. Si la photo est un fichier brut (raw), alors il y aura une case à cocher qui, si elle est cochée, fera que ce style sera appliqué à l'ouverture de toutes les images brutes (raw) prises avec l'appareil spécifiés. Ceci remplacera la Courbe de Tonalité par défaut pour cet appareil.
  6. Choisissez seulement les Outils d'édition de la chaîne de développement que vous souhaitez pour constituer votre style. (Les Outils sont affichés dans le même ordre que dans la chaîne de développement. Il est préférable de nommer les outils, ainsi, chaque outil pourra être identifiés simplement.)
  7. Cliquez Enregistrer.

Note: si vous remplacez Courbe de Tonalité par défault d'un appareil avec votre propre style, vous pouvez le rétablir en choisissant Fichier > Remettre à zéro un Style... et à ce moment, le nom du style.
