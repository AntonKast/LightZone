Enregistrer votre travail

Quand vous enregistrez votre travail, LightZone enregistre votre photo comme un fichier "JPEG LightZone". Un fichier "JPEG LightZone" est un fichier JPEG classique qui est un aperçu d'une version de votre photo mais qui contient également la chaîne de développement que vous avez créée en éditant votre photo. Par défaut, LightZone ajoute _lzn au nom de fichier pour indiquer que c'est un JPEG LightZone.

Étant un fichier JPEG classique, il peut être ouvert par d'autres applications qui gèrent les fichiers JPEG, être envoyer par e-mail ou affiché sur le Web. Lorsqu'il est réouvert avec LightZone, cependant, la chaîne de développement est restaurée et vous pouvez reprendre l'édition là où vous l'aviez arrêtée. Les Vignettes dans le Navigateur Photo distinguent les fichiers JPEG LightZone avec "JPEG-LZN" comme montré ci-dessus.

Un JPEG LightZone se réfère toujours à votre fichier photo original (que ce soit un fichier brut (raw), TIFF, ou un autre JPEG). C'est pourquoi vous ne devez jamais effacer le fichier original de votre photo.

Pour enregistrer votre travail :

  1. Choisissez Fichier > Enregistrer... ou Enregistrer sous...
  2. Entrez un nom pour le fichier.
  3. Choisissez un dossier de destination pour le fichier.
  4. Cliquez Enregistrer.

Note: Autrement, vous pouvez enregistrer votre travail dans un fichier "TIFF LightZone" en modifiant le Paramètre d'enregistrement.
