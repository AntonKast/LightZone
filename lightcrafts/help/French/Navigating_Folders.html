Parcourir les dossiers

Sur le côté gauche de LightZone se trouve Navigateur de Dossiers qui affiche les dossiers sur votre ordinateur. Le Navigateur de Dossiers ressemble et fonctionne comme un Finder Explorateur Windows en mode d'affichage liste.

La différence majeure, toutefois, est que seuls les dossiers et les disques sont affichés -- les fichiers ne l'étant pas. À la place, les vignettes des photos du dossier sélectionnée sont affichées dans la partie Vignettes de la fenêtre.

Le Navigateur de Dossiers vous permet d'accéder à, sélectionner et voir les vignettes et les métadonnées des photos dans les dossiers de votre ordinateur.

Pour accéder à vos dossiers :

  1. Cliquez sur les triangles signes plus '+' pour déplier les dossiers.
  2. Choisissez le dossiers désiré.

De plus, les boutons de navigation peuvent être utilisé.

Le bouton Précédent/Suivant peut être utilisé pour naviguer dans l'historique des dossiers que vous avez consultés précédemment.

Le Bouton Chemin peut être utilisé pour remonter rapidement dans l'arborescence des dossiers.
