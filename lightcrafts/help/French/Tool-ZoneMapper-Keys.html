ZoneMapper : Raccourcis Clavier

L'outil ZoneMapper a un certain nombre de raccourcis clavier qui peuvent être utilisés pour le manipuler. Ils sont indiqués dans le tableau ci-dessous.

Touche Action
tab Choisir le séparateur de zone suivant.
shift-tab Choisir le séparateur de zone précédent.
X Créer un verrou de zone au niveau du séparateur en cours.
Flèches Haut/Bas Déplacer le verrou de zone vers le haut ou vers le bas.
backspace/Suppr. Effacer le verrou sélectionné.

Un Séparateur de zone est la ligne qui divise n'importe quelles zones consécutives. Le séparateur de zone sélectionné est affiché avec une ligne bleue.
