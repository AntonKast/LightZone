Funzione Trascina e rilascia

È possibile trascinare le miniature delle foto dalla sezione miniature della vista Sfoglia e rilasciarle nell'Esploratore cartelle o in altre applicazioni.

Se si trascinano le miniature: Allora:
in una cartella dell'Esploratore cartelle i file immagine saranno spostati
sull'icona di un'applicazione che può aprire file immagine i file immagine saranno aperti
in una cartella o sul desktop i file immagine saranno copiati
in un editor di testo o un emulatore di terminale i percorsi completi dei file saranno incollati

Lo schema è indicativo e non esaustivo poiché ciò che succede quando si rilascia un file immagine sull'icona o la finestra di un'applicazione dipende dall'applicazione stessa.

Per trascinare e rilasciare foto:

  1. Selezionare una o più miniature delle foto che si desidera trascinare e rilasciare: usare il tasto MAIUSC per selezionare più miniature contigue o il tasto COMANDO (su Macintosh) o CTRL (su Windows) per selezionare più miniature non contigue.
  2. Fare clic e trascinare le miniature selezionate nella finestra di un'altra applicazione; rilasciando il pulsante del mouse saranno rilasciate anche le miniature.
