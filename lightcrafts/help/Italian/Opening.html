Come aprire una foto per modificarla

È possibile aprire una foto dalla vista Sfoglia o, se si sta modificando una foto, direttamente dalla vista Modifica.

Per aprire una foto per modificarla:

  1. Selezionare File > Apri...
  2. Cercare la cartella contenente la foto che si desidera aprire.
  3. Selezionare la foto.
  4. Fare clic su Apri.

In alternativa, se la foto è stata modificata di recente, è possibile selezionare File > Apri Recente... e selezionare il nome della foto che si desidera aprire.
