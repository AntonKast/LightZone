Menu

Facendo clic sul pulsante Menu sulla barra del titolo di uno strumento sarà visualizzato un menu a comparsa. Gli elementi del menu sono descritti di seguito.

Attiva/Disattiva
Consente di disattivare o attivare lo strumento. Equivale a fare clic sulla casella di spunta Attiva/Disattiva sulla barra del titolo dello strumento.
Blocca/Sblocca
Consente di bloccare o sbloccare lo strumento. Non è possibile modificare le impostazioni di uno strumento bloccato. Inoltre, non è possibile aggiungere, eliminare, selezionare o modificare le regioni di uno strumento bloccato.
Salva predefinito
Consente di memorizzare le impostazioni correnti per lo strumento.
Applica predefinito
Consente di applicare allo strumento le impostazioni precedentemente memorizzate.
Riduci
Consente di ridurre lo strumento. Equivale a fare doppio clic sulla barra del titolo di uno strumento espanso.
Riduci tutto
Consente di ridurre tutti gli strumenti.
Riduci altri
Consente di ridurre tutti gli strumenti tranne quello corrente.
Espandi
Consente di espandere lo strumento. Equivale a fare doppio clic sulla barra del titolo di uno strumento ridotto.
Espandi tutto
Consente di espandere tutti gli strumenti ridotti.
Espandi automaticamente
Se selezionato, selezionando uno strumento dalla pila strumenti, lo strumento si espanderà automaticamente mentre quello precedentemente selezionato sarà ridotto.
Elimina
Consente di eliminare lo strumento dalla pila strumenti. Equivale a fare clic sul pulsante Elimina sulla barra del titolo di uno strumento.
