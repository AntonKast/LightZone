Styles: Importing

5 of 6

After creating and exporting one or more styles as LightZone Style files (having an lzt extension), they can be imported into LightZone on other computers.

To import a style:

  1. Select File > Manage Styles...
  2. Click Import...
  3. Navigate to the style file you want to import.
  4. Click Choose. (The style will be imported.)
