Opening a photo to edit

You can open a photo from within the Photo Browser or, if you are currently in the Photo Editor, directly from within the Photo Editor.

To open a photo to edit:

  1. Select File > Open...
  2. Navigate to the folder containing the photo you want to open.
  3. Select the photo.
  4. Click Open.

Alternatively, if you have edited a photo recently, select File > Open Recent... and the name of the photo from the menu.
