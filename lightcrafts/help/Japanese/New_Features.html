LightZone の新機能

この項目では LightZone の新機能だけを述べます。主な特徴を知るには、「LightZone を知る」を参照してください。修正されたすべてのバグをチェックするには、github 上の ChangeLogを参照してください。

LightZone 4.1.0

ユーザー インターフェイス

  • 画像を開く際に、コマンドラインや OS のコンテキスト メニューや LightZone のショートカット アイコンへのドラッグ & ドロップが使えるようになりました。
  • Mac OS X:
    • 10.7 (Lion) 以降でフルスクリーン モードをサポートしました。
    • HiDPI retina モードをサポートしました。

写真閲覧ウィンドウ

  • 複数のサムネイルを選択するのに Ctrl-Shift-クリック が使えるようになりました。

写真編集ウィンドウ

  • ズーム イン/アウトが Ctrl-ホイールで、ズームされた画像の位置変更が Ctrl-ドラッグでできるようになりました。
  • 切り抜き/回転モードで、マウス ホイールで画像を回転したり、クリックで回転中心を変更したりできるようになりました。
  • 「情報」タブが編集モードでも見られるようになりました。
  • AdvancedNoiseReduction が V4 に更新されました。
  • 編集後の画像を変換するときにサイズを拡大できるようになりました。

ローカライゼーション

  • UI メッセージがデンマーク語・オランダ語・フランス語・ドイツ語・日本語・スペイン語に翻訳されました。
  • ヘルプ ファイルがデンマーク語・オランダ語・フランス語・日本語・スペイン語に翻訳されました。

OS のサポート

  • Windows (64-bit), Mac OS X 10.10, FreeBSD, OpenIndiana がサポートされました。

LightZone 3.0

ユーザー インターフェイス

  • 閲覧と編集のモードが区別されるようになりました。
  • LightZone 全体でコンテキスト ヘルプが利用できるようになりました。

写真閲覧ウィンドウ

  • 関連する写真をスタックに積み重ねられるようになりました。
  • フォルダー ナビゲーター内のフォルダーへ写真をドラッグ & ドロップして移動できるようになりました。
  • 任意のフォルダー内の写真について最新版のみを選択できるようになりました。
  • IPTC メタデータの編集が可能になりました。

写真編集ウィンドウ

  • すぐ使えるスタイルが付属するようになりました。
  • リライト ツール (トーンマッパー ツールの改良版) に「深さ」と「ファズ」のスライダーが付きました。
  • 履歴タブによって、現在の編集セッションで写真を開いてからの編集履歴をすべて見ることができるようになりました。
  • Raw 調整ツールに「撮影時」ボタンが付きました。
  • LZN ファイルを廃止し、常に JPEG または TIFF ファイルとして作品を保存するようにしたことで、他のアプリケーションとの相互運用性を最大限に高めました。

カメラのサポート

  • Canon EOS 1D Mark III の raw ファイルがサポートされました。
