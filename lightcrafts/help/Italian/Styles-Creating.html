Creazione degli stili

3 di 6

Creare uno stile significa salvare una pila strumenti con nome per poi poterla applicare ad altre foto. È anche possibile sostituire la curva tonale di default di una fotocamera con uno stile creato dall'utente. Ulteriori informazioni sull'applicazione degli stili...

Per creare uno stile:

  1. Creare la pila strumenti che si desidera salvare come stile.
  2. Selezionare File > Salva come Stile...
  3. Dare un nome allo stile.
  4. Dare o scegliere un nome per la cartella dello stile. Gli stili saranno salvati nella cartella scelta dell'utente.
  5. Se la foto è un file immagine raw, sarà disponibile un'opzione che, se selezionata, applicherà lo stile scelto a tutti i file raw della fotocamera specificata. Lo stile sostituirà la curva tonale di default della fotocamera specificata.
  6. È possibile selezionare solo alcuni degli strumenti di modifica dalla pila strumenti e salvarli come stile. Gli strumenti sono mostrati nell'ordine in cui sono stati inseriti nella pila strumenti. Si consiglia di assegnare un'etichetta agli strumenti in modo da identificarli più facilmente.
  7. Fare clic su Salva.

Nota: se si sostituisce la curva tonale di default di una fotocamera con uno stile, è possibile ripristinare la curva tonale di default originale facendo clic su File > Ripristina stile... e il nome dello stile.
