Recadrer une photo

Il arrive que vous souhaitiez recadrer une photo pour enlever une partie non désirée ou pour concentrer l'attention sur le sujet principal. Pour cela vous utiliserez l'outil Recadrage.

Contrairement au autres outils, l'outil Recadrage est spécial. Il est toujours appliqué à la photo entière et n'est pas dans la chaîne de développement.

Pour recadrer votre photo :

  1. Cliquez sur l'outil Recadrage dans la barre d'outils. (Un rectangle blanc de recadrage ainsi qu'une grille apparaîtront au-dessus de votre photo et les contrôles de l'outil recadrage apparaîtront à droite de la barre d'outils
  2. Si vous souhaitez contraindre votre photo à un rapport hauteur/largeur particuliers, cliquez sur le bouton Contraindre les proportions qui représente un cadenas fermé. À ce moment, vous pouvez choisir les proportions d'une de ces deux façons :
    1. Si vous souhaitez un rapport courant, par exemple, 3 x 5, vous pouvez le choisir depuis le menu Proportions courantes.
    2. Si vous souhaitez un rapport personnalisé, vous pouvez entrer vos propres largeur et hauteur dans les deux champs de texte.
  3. Déplacer les côtés ou les coins du rectangle de recadrage là où vous le souhaitez.
  4. Si vous souhaitez déplacer tout le rectangle, vous pouvez cliquer à l'intérieur et alors le déplacer.
  5. Optionnellement, vous pouvez aussi tourner votre photo pour l'aligner. Pour ce faire :
    1. Amener la souris en dehors du rectangle de recadrage. (Le curseur changera pour une icône de rotation.)
    2. Cliquez et déplacez la souris dans la direction vers laquelle vous souaitez tourner votre photo.
  6. Cliquez sur le bouton de la flêche dans la barre d'outils pour soumettre le recadrage. (Le Rectangle de recadrage et les contrôles de recadrage disparaissent et votre photo est recadrée.)
Note: Si lors du recadrage vous souhaitez recommencer du début, vous pouvez cliquer sur le bouton Réinit. (Remettre à zéro le recadrage) de l'outil Recadrage.
