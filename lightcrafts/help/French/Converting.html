Conversion d'Images

1 de 2

Vous pouvez convertir vos photos vers les formats JPEG ou TIFF et ce dans différentes tailles en fonction de vos besoins.

JPEG
Un format d'image qui utilise une compression "avec perte" pour obtenir des fichiers de petite taille. Il est souvent utilisé pour envoyer des photos par E-maiI ou pour les afficher sur le web.
TIFF
Un format d'image qui peut utiliser une compression "sans perte"; cependant, la taille des fichiers est plutôt conséquente. Malgré tout, c'est habituellement le meilleur format à utiliser pour l'import vers d'autres applications.

Pour Convertir votre photo:

  1. Choisissez Fichier > Convertir l'Image... ou Navigateur > Convertir...
  2. Éventuellement entrez un nouveau nom pour l'image convertie.
  3. Choisissez le dossier de destination pour l'image convertie.
  4. Sélectionnez le type d'image pour la conversion : JPEG ou TIFF.
  5. Choisissez le profil couleur à inclure.
  6. Choisissez l'Intention de rendu.
  7. Éventuellement, choisissez l'intention de rendu Colorimétrie Relative avec Compensation du Point Noir.
  8. Éventuellement, définissez la largeur et la hauteur en pixels dans les champs "Redimensionner en" pour réduire votre photo.
  9. Éventuellement, entrer une résolution différente et sélectionner une unité.
  10. Définir la qualité (pour le JPEG) ou la profondeur de couleur (pour les TIFF).
  11. Activez ou non la compression LZW (seulement pour les TIFF).
  12. Cliquer sur Convertir.
