Styles : Exportation

4 de 6

Après la création d'un ou plusieurs styles, vous pouvez les exporter comme des fichiers de Styles LightZone (ayant une extension lzt) qui peuvent être importés dans LightZone sur d'autres ordinateurs.

Pour exporter un style :

  1. Choisissez Fichier > Gérer les Styles...
  2. Choisissez le style que vous souhaitez exporter.
  3. Cliquez sur Exporter...
  4. Naviguez jusqu'au dossier où vous souhaitez exporter le fichier de style.
  5. Cliquez sur Choisir. (Le style sera écrit dans un fichier ayant une extensionlzt.)
  6. Cliquez sur D'accord.
