Blurring a photo

The Blur tool allows you to blur all or parts of your photo by a specified radius. This can be used to soften grainy or blemished areas, or to create out-of-focus areas.

To blur your photo:

  1. Click the blur tool button in the tool bar. (A new blur tool will appear at the top of the tool stack.)
  2. Use the Radius slider to adjust the blurring effect:
    • The Radius slider controls how widely the blurring effect is done around each pixel.
  3. Optionally:
    • Create a mask using regions.
    • Select a different blend mode.
    • Adjust the Opacity slider.
    • Make a color/luminosity selection.
