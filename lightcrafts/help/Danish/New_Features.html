Hvad er nyt i LightZone?

Denne side opremser kun nyhederne LightZone. For at lære om de væsentlige værktøjer, se Discover LightZone siden. For at se alle rettede fejl, se ChangeLogen på github.

LightZone 4.1.0

Brugerflade

  • You can now open an image via command line, OS context menu, or drag & drop on LightZone shortcut icon.
  • On Mac OS X:
    • Full-screen mode is now supported on 10.7 (Lion) or newer.
    • HiDPI retina mode is now supported.

Billed browser

  • Du kan nu vælge flere thumbnails med Ctrl-Shift-Click.

Billed editor

  • Du kan nu zoome ind/od med Ctrl-musehjul, og flytter det zoomede område med Ctrl-træk.
  • I Formindsk/Forstør mode, kan du nu rotere billedet medmuse hjulet, og ændre midtpunktet for roationen med et klik.
  • "Info" tab er nu også tilgængeligt i Editor mode.
  • AdvancedNoiseReduction er opdateret til V4.
  • opskalering er nu muligt når et redigeret billede konverteres.

Localisering

  • UI er oversat til Dansk, Hollandsk, Fransk, Tysk, Japansk, og Spansk.
  • Hjælpe filer er oversat helt eller delvis til Dansk, Hollandsk, Fransk, Japansk, og Spansk.

Operating System Support

  • Windows (64-bit), Mac OS X 10.10, FreeBSD, og OpenIndiana er nu understøttet.

LightZone 3.0

Brugerflade

  • LightZone now has distinct browse and edit modes.
  • There is now context-sensitive help throughout LightZone.

Billed browser

  • Related photos can now be stacked.
  • You can now move photos via drag & drop to folders in the Folder Navigator.
  • You can now select only the most recent versions of your photos in a given folder.
  • You can now edit IPTC metadata.

Billed editor

  • LightZone now comes bundled with many ready-made Styles.
  • The Relight tool (an improved ToneMapper tool) has new Depth and Fuzz sliders.
  • There is now a History tab that shows you all the edits you have make to your photo since opening it for the current editing session.
  • The Raw Adjustments tool has a new As Shot button.
  • LightZone bruger ikke længere LZN files og gemmer dit arbejde som enten JPEG eller TIFF filer for maximal understøttelse af andre programmer.

Understøttede kameraer

  • Canon EOS 1D Mark III raw files are now supported.
