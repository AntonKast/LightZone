Eliminazione dei punti di controllo di una regione

7 di 10

È possibile modificare la forma delle regioni di LightZone in qualsiasi momento. Ciò significa che i punti di controllo che non sono più utili possono essere eliminati.

Per eliminare i punti di controllo di una regione:

  1. Fare clic sul pulsante Regioni nella barra degli strumenti. I pulsanti dei vari tipi di regioni saranno visualizzati a destra della barra degli strumenti; le regioni dello strumento saranno visibili sulla foto.
  2. Per eliminare un punto di controllo, fare doppio clic sulla relativa regione. Saranno visualizzati i punti di controllo.
  3. Fare CTRL-click Clic destro sul punto di controllo che si desidera eliminare: sarà visualizzato un menu a comparsa.
  4. Selezionare Elimina punto.
  5. Ripetere il passaggio numero 3 per ogni punto di controllo che si desidera eliminare.
  6. Una volta regolata l'area di sfumatura, fare clic all'esterno della regione o premere il tasto INVIO o ESC (escape).

Nota: se si dispone di un mouse con più tasti, è possibile aprire il menu a comparsa facendo clic destro anziché CTRL-clic.
