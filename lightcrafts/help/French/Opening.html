Ouvrir une Photo pour l'Éditer

Vous pouvez ouvrir une photo depuis le Navigateur Photo ou, si vous êtes actuellement dans l'Éditeur Photo, directement depuis l'Éditeur Photo.

Pour ouvrir une photo à éditer :

  1. Aller dans Fichier > Ouvrir...
  2. Allez dans le dossier contenant la photo que vous voulez ouvrir.
  3. Choisissez la photo.
  4. Cliquez Ouvrir.

Autrement, si vous avez édité une photo récemment, aller dans Fichier > Ouvrir Fichiers Récents... et choisissez le nom de la photo dans le menu.
