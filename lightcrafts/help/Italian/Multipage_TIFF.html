Immagini TIFF multipagina

Il formato immagine TIFF consente di archiviare più immagini in un unico file TIFF. Questa procedura si usa soprattutto per archiviare pagine digitalizzate di un documento composto da più pagine come i fax.

Per favore l'integrazione con altre applicazioni, LightZone consente all'utente di salvare le foto modificate in un file TIFF di due pagine.

  • La prima pagina è composta dalla foto modificata e dalla pila strumenti di LightZone creata dall'utente.
  • La seconda pagina è composta dalla foto originale non modificata a cui è stata applicata la pila strumenti.

Per default, quando si apre un'immagine TIFF con altre applicazioni, viene mostrata la prima pagina. Perciò la prima pagina è una sorta di anteprima a schermo intero della foto modificata. Quando si apre il file di due pagine in TIFF con LightZone, la pila strumenti viene nuovamente applicata alla foto originale della seconda pagina ed è possibile effettuare ulteriori modifiche.

Uno svantaggio dei file TIFF di due pagine è che sono grandi il doppio. Pertanto si consiglia di usarlo solo per integrare LightZone con Aperture o Lightroom.
