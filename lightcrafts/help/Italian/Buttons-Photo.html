Panoramica dei pulsanti foto

Nella parte superiore sinistra della finestra della vista Sfoglia di LightZone è presente una barra degli strumenti con i pulsanti foto mostrati di seguito.

Il pulsante Modifica equivale a Sfoglia > Modifica.

Il pulsante Apri equivale a File > Apri. Ulteriori informazioni su come aprire una foto per modificarla...

Il pulsante Stampa equivale a File > Stampa. Ulteriori informazioni sulla stampa...

Il pulsante Stili consente di applicare uno stile alla foto la cui miniatura è selezionata. Ulteriori informazioni sugli stili...

Il pulsante Invia equivale a File > Invia. Ulteriori informazioni su come inviare foto...

Il pulsante Converti equivale a File > Converti. Ulteriori informazioni sulla conversione di un'immagine...
