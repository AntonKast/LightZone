Conversione di un'immagine

1 di 2

È possibile convertire le foto in immagini JPEG o TIFF e in vari formati adatti a scopi differenti.

JPEG
Questo formato immagine si ottiene tramite una compressione "con perdita" che produce file di piccole dimensioni. Spesso è usato per inviare foto via e-mail o per caricarle sul web.
TIFF
Questo formato immagine si ottiene tramite una compressione "senza perdita" ma i file sono di grandi dimensioni. Nonostante ciò, questo tende a essere il formato più adatto all'importazione in altre applicazioni.

Per convertire le foto:

  1. Selezionare File > Converti immagine... o Sfoglia > Converti...
  2. È possibile scegliere un nuovo nome per l'immagine convertita.
  3. Selezionare una cartella di destinazione per le immagini convertite.
  4. Selezionare il tipo di immagine in cui convertire: JPEG o TIFF.
  5. Selezionare il profilo colore da includere.
  6. Selezionare l'Intento rendering.
  7. È possibile selezionare Compensazione del punto nero (si applica solo quando l'intento rendering è Colorimetrico relativo).
  8. È possibile ridurre la foto immettendo una larghezza e un'altezza in pixel nei campi di testo di Ridimensiona.
  9. È possibile scegliere una risoluzione differente e selezionare un'unità.
  10. Impostare la qualità (per JPEG) o la profondità di bit (per TIFF).
  11. Se si desidera, scegliere la compressione LZW (solo per TIFF).
  12. Fare clic cu Converti.
